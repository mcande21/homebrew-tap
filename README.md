# homebrew-tap

Homebrew tap for mcande21 tools.

## Install

```bash
brew tap mcande21/tap
brew install openclaw-bridge
```

## Formulae

| Formula | Description |
|---------|-------------|
| `openclaw-bridge` | CLI bridge connecting Claude Code to OpenClaw gateways |

## Usage

After installing `openclaw-bridge`, the `ocb` binary is available on your PATH:

```bash
ocb --help
```
